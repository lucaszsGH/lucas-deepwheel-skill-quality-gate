# 新用户能力体检

## 目标

避免用户误以为“装了 Skill 就具备全部能力”。首次使用或发布前，必须说明当前环境具备什么、缺什么、缺了如何降级。

## 必查项

```text
文件写入目录
网页抓取能力
受限来源处理
PDF 文本提取
扫描 PDF OCR
图片 OCR
视频 / 音频转录
字幕 / 转录稿处理
外发通道
下游复现生成能力
```

## 输出格式

```text
当前可直接做：
当前需要工具/权限：
当前不承诺：
建议先做的最小测试：
是否需要安装或启用工具：是/否，安装前需确认
```

## 安装引导要求

安装或启用任何工具前必须告诉用户：

```text
安装什么
为什么需要
不安装如何降级
安装后如何验收
```
